ArtNet Node 2x

Art-Net → DMX конвертер с двумя независимыми DMX выходами и web интерфейсом

Описание

ArtNet Node 2x — профессиональное устройство для преобразования Art-Net сигнала в два независимых DMX512 выхода. Устройство работает по Ethernet и имеет удобный web-интерфейс для настройки параметров.
